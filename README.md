# Slay the Gray — Homepage

Public homepage for [Slay the Gray](https://github.com/esattler/slay-the-gray), a mobile app for miniature painters to track their collection, log painting sessions, and conquer their Pile of Shame.

Hosted via GitHub Pages.

## Pages

| Page | Path | Purpose |
|------|------|---------|
| Homepage | `/` | App overview and feature summary |
| Privacy Policy | `/privacy.html` | Privacy policy (required for Google OAuth verification) |
| Terms of Service | `/terms.html` | Terms of service |

## Deployment

This site is deployed automatically via GitHub Pages from the `main` branch root.

Enable via: **Settings → Pages → Source → Deploy from branch → `main` / `/ (root)`**
